# showcase-shop_smol

### Описание

Маленький шоукейс api с товарами их свойствами и возможностью фильтрации по свойствам.

### --- quickstart ---

    git clone https://github.com/Nhemas/showcase-shop_smol.git
    cd showcase-shop_smol
    composer create
    php artisan db:seed
    php artisan serve
